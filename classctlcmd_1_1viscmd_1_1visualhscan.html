ctlcmd.viscmd.visualhscan Class Reference
Command Line Interface Manual » SiPM Analysis-Level commands

Detailed Description

This command assumes that at most a single detector element will be visible to the visual system at one time. The command then stores the gantry coordinates and the found detector center in pixel cooridnates together, and this is used to create the transformation matrix between visual coordinates and gantry coordinates for fast visual calibration.
