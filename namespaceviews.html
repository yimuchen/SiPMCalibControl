Detailed Description

Response to be submitted for URL requests, there are 2 types of calls that we will consider:

  • Request for an HTML page. This is implemented as jinja templates found in the server/template directory.
  • Request for non-time critical data, called for via AJAX requests. This is implemented by converting python dictionary to a JSON format via the flask.jsonify method.

As flasks add_url_rule method requires a callable with no other inputs other than the URL, we use a thin wrapper class to allow all function to access the main session instance.
