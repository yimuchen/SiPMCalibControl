ctlcmd.getset.runfile Class Reference

Detailed Description

Notice that while runfiles can be called recursively, you cannot call runfiles that have already been called, as this will cause infinite recursion. If any command in the command file fails, the whole runfile call will be terminated to prevent user error from damaging the gantry.
