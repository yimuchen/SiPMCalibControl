ctlcmd.cmdbase.controlsignalhandle Class Reference

Detailed Description

In this case, store that the termination signal has been requested, and do nothing else, the program should handle how to gracefully terminate the current running progress. Solution is taken from: https://stackoverflow.com/questions/18499497/how-to-process-sigterm-signal-gracefully

Storing the original in-built functions defined by python used to handle the interrupt and terminate signals.
