The argument parser class is initialized with the class name as the program name and the class doc string as the description string. This greatly reduces the verbosity of writing custom commands. Each command will have accession to the cmd session, and by extension, every interface object the could potentially be used.

ctlcmd.analysiscmd.zscan.run (   self,
  args 
)

For each of the z values and PWM power value listed in the arguments, we move the gantry over to the cooridnates, set the PWM settings, and take a measurement. The measurement is then saved to the standard data format. As there are no fitting done here, the command simply exists once all data collection is complete.
