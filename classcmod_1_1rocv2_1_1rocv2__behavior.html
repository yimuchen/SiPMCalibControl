cmod.rocv2.rocv2_behavior Class Reference

Detailed Description

This includes following parts:

  • Expanding the per-half variables to effectively be per-channel (easier mapping of variables while keeping the data small).
  • Expanding the raw channel indices values to something that is easier to handle for the analysis (no overlap per event)
  • The addition of the channel type variable.
