cmod.fmt.FIFOHandler Class Reference

Detailed Description

As we don't expect to flush the log unless explicitly requested by the user, here we provide a class that stores the logging records in a first-in-first-out list, so that logging information can be retrieve for later use. The user should specify a maximum capacity.
