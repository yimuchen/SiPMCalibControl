views.FileDataParsing.make_hist_processed_data (   self,
  filename 
)

Here we assume the standard write-out format with the last N columns being a list of readout values. The first line is then used to create an numpy histogram. All other lines will be accumulated into this histogram. We choose to create the histogram early to reduce the memory footprint of the program.
