cmod.board.Detector Class Reference

Detailed Description

The handling of the detector. No name will be given for the detector element here, that will be handled by the parent "Board" class.

The calibrated coordinates (either the visually calibrated coordinates and the accompanying transformation matrix, or the luminosity aligned coordinates will be stored as a dictionary, with the z operation value used for obtaining the calibration used as the key.)
